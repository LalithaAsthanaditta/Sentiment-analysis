# Sentiment-analysis
Sentiment Analysis project that classifies text as positive, negative, or neutral using machine learning. Includes data preprocessing, model training, and prediction scripts.

This project demonstrates the end-to-end process of building a natural language processing (NLP) application, including:

Text preprocessing with lemmatization and removal of stopwords for cleaner input.

Feature extraction using TF-IDF vectorizer to capture the importance of words.

Sentiment classification using a trained Logistic Regression model.

A sleek, modern UI for easy input and instant sentiment feedback.

A history feature to save and display past predictions with timestamps and visualize sentiment distribution.

This app is perfect for beginners to understand NLP concepts and for anyone interested in analyzing text sentiment quickly and accurately.
